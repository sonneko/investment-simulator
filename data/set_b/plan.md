# 投資計画（2026年6月・最終調整）

## 基本方針
- AI・半導体およびテックグロース株への集中投資による最大リターンの追求。
- 最終セッションに向けてキャッシュ比率を下げ、株式および仮想通貨への投資比率を極限まで高めてリターンを最大化する。

## アセットアロケーション目標
- NVDA（AI半導体本命）: 40%（現状維持）
- QQQ（テックグロース）: 35%（引き上げ）
- BTC-USD（仮想通貨）: 10%（目標維持）
- キャッシュ（機動枠）: 15%（引き下げ）

## リスク管理
- 損切りルール（-10%）を徹底し、急な市場の反転に備える。最終局面での利益確定のタイミングを逃さない。
