# 投資計画（最終総括）

本シミュレーションの最終セッション（3/3）において、当初の「資本保全を最優先とする」方針を徹底し、追加の取引を行わずに現在のポートフォリオを維持して運用を完了します。

### 最終資産配分（現状維持）
- **米国株式 (VTI)**: 長期的な成長の果実を享受
- **米国債券 (AGG/BND)**: ポートフォリオのクッションとして機能
- **ゴールド (GLD)**: インフレ・地政学リスクへのヘッジ
- **キャッシュ (USD)**: 高い流動性と安全性を確保（約30%以上）

極めて強固で防御力の高いポートフォリオを維持できたことで、マクロ経済の不確実性や金利変動リスクから資産を守り抜き、無駄な取引コストを抑えながら、安定した運用成果を達成しました。元本の維持とインフレ率を上回るリターンという目標を十分に達成可能な状態でシミュレーションを終了します。
